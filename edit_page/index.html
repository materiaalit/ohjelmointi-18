Näin lähetät muutosehdotuksen materiaaliin

Voit lähettää muutosehdotuksen kurssin Github-projekiin. Github on yleinen palvelu, jossa voi hallinnoida ja jakaa muille projekteja, ohjelmistoja ja lähdekoodia.

Muutosehdotusten lähettäminen vaatii käyttäjätunnukset Githubiin. Jos sinulla ei ole vielä Github-tunnusta, voit luoda sen täältä.

Ennen kuin lähetät muutosehdotuksesi, katsothan täältä ettei joku muu ole jo lähettänyt samaa muutosehdotusta.

Lähetä muutosehdotus näin

  1. Paina yllä olevaa nappia muokataksesi sivua, jolla olit äsken.
  2. Vaihda muokkausikkunan oikeasta yläkulmasta "No wrap" --> "Soft wrap".
  3. Etsi materiaalin kohta, johon haluat tehdä muutoksen. Voit hakea sopivaa kohtaa painamalla näppäinyhdistelmää Ctrl+f ja kirjoittamalla hakukenttään sopivan hakutermin.
  4. Selaa sivun alareunaan, ja kirjoita osioon "Propose file change" lyhyt ja ytimekäs otsikko, esimerkiksi "Kappaleen 2 kirjoitusvirheen korjaus".
  5. Paina vihreää nappia "Propose file change".
  6. Paina avautuvalta sivulta nappia "New Pull Request".
